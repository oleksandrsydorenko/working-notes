# Hotkeys Plugins

## Code Runner

Run : ⌘ + ⌥ + N
Stop : ⌃ + ⌥ + M

## Live Sever

Start : ⌘ + L, ⌘ + O
Stop : ⌘ + L, ⌘ + C

## Markdown All in One

Open Preview : ⌘ + ⇧ + V

## Notes

List Notes : ⌥ + L

## Regex Previewer

Open Preview : ⌘ + ⌥ + M

## REST Client

Send Request : ⌘ + ⌥ + R

## Settings Sync

Upload Key : ⇧ + ⌥ + U
Download Key : ⇧ + ⌥ + D

## Turbo Console Log

Insert Log : ⌥ + ⌃ + L
