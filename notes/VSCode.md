# VSCode

## Command Palette

Show Command Palette: `⌘ + ⇧ + P / F1`

## Editor

Fold: ` ⌘ + ⌥ + [`\
Unfold: `⌘ + ⌥ + ]`

## GoTo

Go To File: `⌘ + p`\
Go To Line: `⌘ + p , ⇧ +:`

## Sidebar

Open/Close Sidebar: `⌘ + B`\

## Terminal

Open/Add Terminal: `⌃ + ⇧ + ~`\
Close/Remove Terminal: `⌃ + ⇧ + X`
